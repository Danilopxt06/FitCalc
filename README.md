# 💪 FitCalc

## 📖 Sobre o projeto

O **FitCalc** é uma plataforma digital desenvolvida para auxiliar pessoas que desejam adotar um estilo de vida mais saudável e equilibrado.

Através de uma interface simples, intuitiva e acessível, o sistema permite ao usuário calcular seu **Índice de Massa Corporal (IMC)** de forma rápida e prática. Com base nos dados informados, a plataforma realiza uma análise do perfil corporal e apresenta uma **classificação detalhada do estado nutricional**, ajudando o usuário a compreender melhor sua condição física.

Além disso, o **FitCalc** oferece **recomendações de treinos organizados por grupo muscular**, servindo como apoio para quem busca iniciar ou aprimorar sua rotina de exercícios.

---

## ✨ Funcionalidades

✔️ Cálculo automático do **Índice de Massa Corporal (IMC)**  
✔️ Classificação detalhada do estado nutricional  
✔️ Recomendaações de treinos por grupo muscular  
✔️ Interface simples, responsiva e intuitiva  
✔️ Navegação prática e acessível  

---

## 🎯 Objetivo

O projeto tem como principal objetivo **promover a conscientização sobre saúde e bem-estar**, fornecendo informações úteis que incentivem o acompanhamento físico e a adoção de hábitos mais saudáveis.

2. Clique para calcular seu IMC  
3. Visualize sua classificação corporal  
4. Confira as recomendações de treino sugeridas

## 💡 Propósito

Mais do que uma calculadora, o **FitCalc** é uma ferramenta educativa criada para incentivar hábitos saudáveis e demonstrar, na prática, como a tecnologia pode contribuir para a qualidade de vida.
